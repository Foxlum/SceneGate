    SceneGate

    Work-in-progress tool for reverse engineering, file format analysis, modding and localization.

    Features

    • Navigate the virtual file system from Yarhl.
    • Convert files with Yarhl converters via plugins.

    Yarhly shipped plugins

    • Lemon: 3DS formats

    Installation

    Install the Material Icons font from here.

    The project ships the application as a portable ZIP file that does not require any installation. Just unzip and run!

    Build

    The project requires to build .NET 5.0 SDK and .NET Core 3.1 runtime. If you open the project with VS Code and you did install the VS Code Remote Containers extension, you can have an already pre-configured development environment with Docker or Podman.

    To build, test and generate artifacts run:

    # Only required the first time
dotnet tool restore

# Default target is Stage-Artifacts
dotnet cake

    To just build and test quickly, run:

    dotnet cake --target=BuildTest
